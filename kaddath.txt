json struct for file:
path, raw text, db

for phrase:
filepath, raw phrase, num syllables, phoneme list,

azathoth
* listen at some port
* accept json data terminated by PROSAICFHTAGN
* run through tokenizer/phraser and into mongodb

nyarlarthotep
* point at file or directory
* (recursively) read and send each file in a json obj off to azathoth
* spec a target db

cthulhu
* accept poem structure and db name (default to huge bog db)
* read from mongodb
* output poem in text
